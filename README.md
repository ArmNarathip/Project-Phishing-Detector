# Phishing Mail Detector (Chrome Extension)

## Install (Load unpacked)
1. Open Chrome -> chrome://extensions
2. Enable Developer mode
3. Click "Load unpacked"
4. Select this folder (phishing-mail-detector)

## How to use
1. Open Gmail / Outlook Web
2. Open an email message
3. Click the extension icon -> Scan
4. It will open Result page with Normal/Suspicious/Dangerous

## VirusTotal
- Go to Options page
- Paste your VirusTotal API key
- Use "VirusTotal URL Check" on Result page

> Note: Webmail DOM can change anytime, selectors may need tweaking per UI updates.
